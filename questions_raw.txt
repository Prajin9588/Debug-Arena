Q1 — Print Statement Typo

Broken Code

prnt("Hello, Swift!")


Error
Cannot find 'prnt' in scope

Correct Code

print("Hello, Swift!")


Riddle
I speak to the console, miss me and nothing prints.

Hidden Test Cases (Logic-validated)

print("Test") → "Test"
print("123") → "123"
print("") → ""


Logic Rule
Use the correct built-in function for console output.

Regex / Token Rules

^print\s*\(.*\)$


Answer
Correct function name is print.
